# GIIP Project

## Overview
This project is a web application that provides data visualization and analysis features. It includes multiple modules such as dashboard, country analysis, reports, and feedback.

## Features
- Dashboard view
- Country analysis
- Interactive pages
- User feedback system
- Reports section

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Project Structure
- frontend/ → contains all UI pages
- backend/ → contains server, routes, and services

## How to Run

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Open the HTML files directly in your browser
